# EFI-11Gen-Intel
EFI for Big Sur Hackintosh for Tiger Lake Intel CPU

This EFI works with Big Sur on Tiger Lake CPU. Tested on HP- FR2005TU. 
Not Working:
Trackpad
Camera
Microphone
Battery (not shown)
Graphics
SSD (not detected)
Audio
